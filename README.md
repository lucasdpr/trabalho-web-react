Sistema de Login e Cadastro (Trabalho Web) 
Projeto desenvolvido para a nota 1 da disciplina de Desenvolvimento Web, focado em autenticação via Google e manipulação de objetos JSON.

 🛠️ Tecnologias Utilizadas
React.js: Uso de componentes funcionais, useState e useEffect.
React Router: Gestão de navegação entre as páginas.
Google OAuth: Autenticação segura de usuários. 
JWT-Decode: Tratamento dos dados recebidos do Google (foto, nome e e-mail). 
Vercel: Plataforma utilizada para o deploy e hospedagem.
📌 Funcionalidades do Projeto
Home: Tela de entrada com login social que captura e exibe os dados do perfil logado. 
Apresentação: Espaço com informações sobre o desenvolvedor e o curso.
Cadastro: Formulário inteligente que pré-preenche nome e e-mail vindos do Google e gera um objeto JSON final. 
⌨️ Como rodar o código localmente
Faça o clone deste repositório.
No terminal, digite npm install.
Inicie com npm start.
